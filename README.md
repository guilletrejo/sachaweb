This is a personal project created to start learning fundamentals of backend development.

It is intended for software/systems/computer/whatever engineers that may have experience with programming in different languages but not with these tools and concepts.

I'm using AI Agents to code it. I think the important part is to understand WHY things are done in a certain way. Knowing the syntax and memorizing templates is not as important now as it was in the past.

I'm asking the AI to leave ultra-redundant and ELI5 comments if needed to illustrate and explain every part of the code.
